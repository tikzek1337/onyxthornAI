# Dataset format for onyxLiteAI

JSONL: одна строка — одна обучающая запись.

```json
{"instruction":"Пользовательский запрос","input":"Необязательный контекст","output":"Желаемый ответ ассистента"}
```

В процессе подготовки запись превращается в chat-format:

```text
<|bos|><|system|>
...system prompt...<|end|>
<|user|>
...instruction + input...<|end|>
<|assistant|>
...output...<|end|><|eos|>
```

## Качество данных

Хорошая запись для onyxLiteAI:

- звучит как нормальный человеческий диалог;
- не делает вид, что знает неизвестные факты;
- задает уточняющий вопрос, если данных мало;
- не превращается в coding answer;
- безопасна в медицинских, юридических, финансовых и кризисных темах;
- содержит конкретный следующий шаг, если пользователь просит помощи.

## Что не добавлять

- большие куски кода;
- инструкции по взлому, обходу правил или вреду;
- персональные данные реальных людей;
- устаревающие факты без даты;
- ответы с чрезмерной уверенностью там, где нужна осторожность.


## Entertainment-записи

В версии 0.3.0 добавлены spoiler-light записи про фильмы, сериалы и аниме. Они нужны не для энциклопедической точности по текущим релизам, а для обычного разговора: вайб, темы, кому может зайти, что выбрать вечером, как обсудить после просмотра.
