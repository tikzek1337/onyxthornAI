# data/raw

Клади сюда свои дополнительные JSONL-файлы для onyxLiteAI.

Главное правило: onyxLiteAI — модель для обычного общения, а не для кодинга. Не добавляй сюда большие массивы кода, debugging-задачи и programming tutorials, если хочешь сохранить chat-only характер модели.

Хорошие данные для дообучения:

- реальные диалоги поддержки без персональных данных;
- объяснения бытовых, учебных и общекультурных тем;
- разговорные записи про фильмы, сериалы и аниме без спойлеров;
- сообщения в разном тоне: мягче, увереннее, короче;
- примеры честного “я не знаю”;
- безопасные ответы в медицинских, юридических и финансовых темах;
- ответы, где модель задает короткий уточняющий вопрос.

Формат записи:

```json
{"instruction":"Привет, поболтай со мной","input":"","output":"Привет. Можем просто спокойно поговорить..."}
```
